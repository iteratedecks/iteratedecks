Create text file with name "Include.txt" to use it in wildcard include specific filter.
Create text file with name "Exclude.txt" to use it in wildcard exclude specific filter.
Both files have the same format - one card name in each line, you can access cards by ID using square brackets, example of Include.txt file:

// This is the include.txt file
// Only ONE card per line, no separators
Disposer
Kogoth the Consumer
Abominable Raksha
Hephatat[539]
Hephatat[537]
